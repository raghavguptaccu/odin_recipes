# odin_recipes

##Odin Recipes

This project is a simple recipe website built as part of The Odin Project. It will contain a main index page with links to a few recipe pages and will help reinforce the basics of building and structuring a website with HTML.

By completing this project, I will have demonstrated an understanding of basic HTML structure, working with links and images, organizing content across multiple pages, and using Git and GitHub to manage a small project.
